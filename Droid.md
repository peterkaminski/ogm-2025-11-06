# Droid

> **Note**: This agentic AI tool was mentioned by [[Pete Kaminski]] alongside [[Claude Code]] during the OGM 2025-11-06 call.

## What is Droid?

**Droid** is an agentic AI coding assistant created by Factory Eye that competes with [[Claude Code]] and similar tools.

## Features

- Web-based interface
- Works on mobile devices
- Agentic capabilities (autonomous task completion)
- Can read and write multiple files
- Creates and manages task lists

## Mentioned in Context

Pete discussed it as:
- Alternative to Claude Code
- Also has web version for accessibility
- Running a "big promotion" at the time of the call
- Part of the broader "everybody's gonna be doing this in 18-24 months" prediction

## Factory Eye

The company behind Droid, focused on AI-powered development tools.
